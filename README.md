# React-TODO-app

    This is a simple TODO app built with React.

  ## Features
      Add new tasks
      Mark tasks as completed
      Delete tasks
      Filter tasks by completion status
  
  ## Technologies Used
      React
      JavaScript
      TypeScript
      RESTful API
      SCSS
